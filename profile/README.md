## Projects

### Limen *(in development)*

Limen is a portable, contract-enforcing computation graph framework for AI-enabled embedded systems.

It enables the same system definition to run across:
- bare-metal microcontrollers (`no_std`)
- embedded Linux devices
- multi-threaded servers

Key properties:
- zero dynamic dispatch in the hot path  
- contract enforcement (freshness, liveness, deadlines)  
- inference as a first-class system component  
- compile-time graph generation  

→ Repository will be made public ahead of v0.1.0

### Zcash Ecosystem Contributions

Systems-level work within the Zcash ecosystem, in collaboration with ZingoLabs and the wider Zcash developer community.

This includes:

- [Zaino](https://github.com/zingolabs/zaino) — a Rust-based indexing service forming part of the Z³ transition (Zebra, Zaino, Zallet), supporting the migration away from legacy `zcashd` infrastructure  
- Contributions to [Zebra](https://github.com/ZcashFoundation/zebra), the Rust full node implementation developed by the [Zcash Foundation](https://github.com/ZcashFoundation)  
- Early work on **Zcash over Nym**, exploring network-level privacy  
- Contributions to the emerging full-node wallet ([Zallet](https://github.com/zcash/wallet))  

Zaino is developed by [ZingoLabs](https://github.com/zingolabs) as part of a broader ecosystem effort to modernise Zcash infrastructure, transitioning from legacy C++ systems to a modular Rust-based stack.
